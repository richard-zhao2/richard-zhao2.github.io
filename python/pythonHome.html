Computer Science with Python

Hi, my name's Richard and this is a very simple introduction to computer science.

One of the main issues I have with a lot of online resources is that they're usually glorified syntax references. Rarely do I find sources that develop problem-solving skills, and the ones that do often focus very heavily on that alone, so I made this site in hopes that it'll help bridge the gap between knowing a bunch of syntax and knowing how to actually program.

I like starting off with Python because it's an easy language to pickup, but language doesn't really matter in the end.
